# @developjik1/icons

SVG 아이콘 라이브러리에 오신 것을 환영합니다!

## 설치

```bash
npm install @developjik1/icons
```

## 사용법

```tsx
import { Home, Heart, Search } from '@developjik1/icons';

function App() {
  return (
    <div>
      <Home size={24} color="blue" />
      <Heart size={32} />
      <Search className="search-icon" />
    </div>
  );
}
```
 
## 특징

- 🎨 **15개의 고품질 아이콘**
- ⚡ **경량화된 SVG 기반**
- 🔧 **커스터마이징 가능** (크기, 색상, 클래스)
- 📦 **Tree-shaking 지원**
- 🔍 **TypeScript 완전 지원**
- ⚛️ **React 17+ 호환**
